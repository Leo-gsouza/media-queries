Estilos em uma unica declaração

Criar o estilo geral, retrato e paisagem no proprio HTML usando a tag style com @media para os tipos e caracteristicas diferentes. Exemplo abaixo

Dentro da TAG style no HTML ou em um unico estilo CSS /*Declarações Retrato*/ @media screen and (orientation: portrait) { body { background: url('jpeg/cev-portrait.jpg') no-repeat center bottom ; background-size: contain; } } /*Declarações Paisagem*/ @media screen and (orientation: landscape) { body { background: url('jpeg/cev-landscape.jpg') no-repeat left bottom; background-size: contain; } }
